## hz-node-packages
Repository that allows publishing packages for multiples internal apps

## List of packages
- /types: package for saving all the different types across multiple apps made by myself

## How to work in this repo?


If you want to make changes remember we have changesets

1) make your local changes
2) execute ```npx changeset```
3) commit and push the changes
4) execute ```npx changeset version```
5) merge and then publish
